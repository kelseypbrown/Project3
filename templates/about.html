About our Project

Original dataset used: "Most Streamed Spotify Songs 2023" by Nidula Elgiriyewithana sourced from Kaggle

When analyzing the data from Spotifys most streamed songs for 2023, we wanted to ask ourselves a few questions:

- Which data should be analyzed (what is being kept vs cleaned)?

Because we specifically wanted to look at Spotify, data regarding Apple Music, Shazam, and Deezer were dropped when cleaning our set. Genres were added to the cleaned dataset using Spotipy for Spotify API, and narrowing down genre categories further to lessen the amount of unique genres. Only the first genre for each artist and song was kept.

- Why assign genres to this streaming information using Spotipy for Spotify API?

We found it more accurate to work with a Spotify API source than to use another dataset to determine genres. Using the genre information assigned to the top songs data, we can look at the most popular genres, top songs in each genre, top artists per genre, etc.

- How can this data be used in real life?

Spotify uses this data in its yearly promotion “Spotify Wrapped”, in which users and producers can see their own and others listening trends. At the end of the year, users are encouraged to view their data and share it via social media like Instagram and Facebook. In addition, producers and management teams can use this data to make decisions on which artists to promote or tour, Spotify can use it to build playlists for the app or viewing usage and streaming trends, and scouts can use it to discover new talent according to said trends, etc. Competitor apps can use this information as well to determine new promotions, user sign-up incentives, and user streaming preferences. With data like keys and BPM, producers can locate trends in production to see what might be working and what they may need to change when creating with musicians.
